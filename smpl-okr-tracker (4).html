Como Definir Bons OKRs
A estrutura
Objetivo (O)
Onde queremos chegar. Qualitativo, inspirador e temporal.
Key Result (KR)
Como sabemos que chegamos lá. Quantitativo, mensurável. 2–4 por objetivo.
Iniciativa
O que faremos concretamente para mover cada KR.

Critérios para bons objetivos
  • Inspirador: motiva o time a agir
  • Qualitativo: sem números (eles ficam no KR)
  • Temporal: claro para qual ciclo se aplica
  • Alinhado à estratégia da empresa
  • Desafiador mas alcançável com esforço real
Critérios SMART para Key Results
  • Specific — específico e claro
  • Measurable — sempre com número
  • Achievable — desafiador mas possível
  • Relevant — relevante para o objetivo-pai
  • Time-bound — prazo definido
